HTML

html:5 lub !

Wystarczy wpisać jedno z tych poleceń, aby wygenerować kod początkowy strony.

a:mail

Tworzy hiperłącze mailowe.

link:css

W ten sposób możemy dołączyć plik .css o standardowej nazwie 'style' (którą oczywiście można zmienić).

form:post lub form:get

Tworzy formularz typu POST lub GET.

CSS

pos

Skrót dla position (domyślnie z wartością relative). Po dwukropku możemy dopisać skrót parametru (np. pos:a -> position:absolute).

c

Skrót dla color (domyślnie z wartością #000).

cu lub cr lub cur

Skrót dla cursor (domyślnie pointer). Można wybrać, który chcemy po dwukropku (np. cr:hr -> cursor:help).

m:a

Skrót do automatycznego marginesu.

fs

Skrót dla font-style (domyślnie italic).

ff

Skrót dla font-family (domyślnie serif).
